# Retro Checklist

Purpose: Ensure retros create actionable outcomes and follow-ups.

Pre-retro
- [ ] Schedule and invite stakeholders
- [ ] Prepare timeline and data (incidents, metrics, highlights)
- [ ] Share agenda and pre-reading 48 hours before

During retro
- [ ] Start with clear goal
- [ ] Capture positives, deltas, and ideas (use templates)
- [ ] Prioritize top 2–3 actions with owners and due dates
- [ ] Confirm who will track each action

Post-retro
- [ ] Publish notes to docs/retros/ with actions and owners
- [ ] Create issues for each action with the retro as context
- [ ] Review progress on previous retro actions at next retro

Facilitation tips
- Timebox activities
- Use a neutral facilitator for candid feedback
- Focus on concrete, small, testable changes
